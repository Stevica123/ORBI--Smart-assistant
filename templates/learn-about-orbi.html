Discover Orbi

Orbi uses MongoDB Atlas as a cloud-based NoSQL database to store its knowledge. Within the knowledge_db collection, data is structured as qa_pairs—questions and answers that the assistant uses to respond quickly and accurately. This setup allows for easy updates, analysis, and learning, with secure and real-time access.

Orbi is powered by the ChatGPT 3.5 Turbo model, a state-of-the-art language model developed by OpenAI. Using the OpenAI API, Orbi processes user questions, understands context, and generates fast, natural, and helpful responses in real time—making the assistant intelligent, responsive, and easy to interact with.

Orbi’ s Interface Pipeline

Spline was used to create and customize 3D elements that enhance the visual experience of Orbi. These models were integrated directly into the Figma layout to give the assistant a more dynamic and modern look, all while keeping performance in mind.

Anima was used to turn static designs into interactive prototypes and export production-ready code. It allowed me to simulate real user flows directly from Figma, adding animations, transitions, and responsive behavior—all without writing code during the design phase.

The entire design of Orbi was created in Figma, enabling a clean, intuitive, and consistent user interface. Figma allowed for quick iterations, real-time adjustments, and a streamlined design process focused on clarity, usability, and visual harmony.

Driven by curiosity, guided by simplicity, and built to make technology feel more human. Bringing clarity and confidence to your digital world—one interaction at a time.
